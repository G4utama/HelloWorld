# Hello, World!

## Assembly
```
nasm -f elf64 -o HelloWorld.o HelloWorld.asm && ld HelloWorld.o -o HelloWorld && ./HelloWorld
```

## Brainfuck
```
bf HelloWorld.bf
```

## C
```
gcc HelloWorld.c -o HelloWorld.out && ./HelloWorld.out
```

## C++
```
g++ HelloWorld.cpp -o HelloWorld.out && ./HelloWorld.out
```

## Java
```
javac HelloWorld.java && java HelloWorld
```

## Pyhton
```
python3 HelloWorld.py
```
